# Data Deletion Instructions for iConnectV2

Facebook requires apps to provide a simple process for users to request the deletion of their data.  
If you have previously logged in to iConnectV2 using Facebook Login and wish to delete your data, please follow the instructions below.

## 1. Automatic Data Deletion (In-App)
If the iConnectV2 app contains a “Delete My Data” option, using that feature will remove all personal data stored on our servers.

## 2. Manual Data Deletion Request
If you want your Facebook-related data deleted manually, please send an email to:

**support@tekchant.com**

Your request must include:
- Your full name
- Facebook User ID (if known)
- Email or phone number linked to your account

We will delete all associated data within **30 days** as per Facebook Platform Policy.

## 3. What Data Do We Store?
When you log in using Facebook, we may store:
- Name
- Email
- Profile picture URL
- Facebook ID
- Access token (temporary)

No other Facebook data is stored.

## 4. Confirmation
After processing your deletion request, we will send a confirmation email once your data has been removed completely.

## 5. Contact
For any data privacy concerns or questions, reach us at:

**Email:** support@tekchant.com  
**Website:** https://www.tekchant.com
